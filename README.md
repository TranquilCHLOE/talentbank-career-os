# AI Career Analyzer

An AI-powered career analysis platform built with Next.js, Groq AI, and Supabase.

## Features

* Resume analysis using AI
* Career recommendations
* Skill matching
* Salary benchmarking
* Industry demand insights
* Learning roadmaps
* Portfolio history storage

## Tech Stack

* Next.js
* TypeScript
* Supabase
* Groq API
* Tailwind CSS

## Database

* careers
* skills
* career_skills
* learning_roadmaps
* salary_benchmarks
* industry_demand
* portfolios

## Setup

1. Clone repository
2. Install dependencies

```bash
npm install
```

3. Create `.env.local`

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
GROQ_API_KEY=
```

4. Run

```bash
npm run dev
```
